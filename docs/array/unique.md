# unique

Creates an array of the unique values of given array

`Since v1.0.0`

## Arguments

- `arr: T[]` - Array of primitive values

## Returns

`T[]` - Array of unique values

## Usage

```ts
import * as _ from "litus";

_.unique([3, 3, 1, 2, 2, 3]);
// => [3, 1, 2]
```
