# max

Finds the element with the maximum value

## Arguments

- `arr: T[]`
- `fn: (val: T) => number`

## Returns

`T`

## Usage

```ts
import * as _ from "litus";

const people = [
  { age: 18, name: "John" },
  { age: 30, name: "Jane" }
];

_.max(people, (el) => el.age);
// => { age: 30, name: "Jane" }
```
