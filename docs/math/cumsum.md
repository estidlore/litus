# cumsum

Calculates the cumulative sum of an array of numbers

## Arguments

- `arr: number[]`

## Returns

- `number[]`

## Usage

```ts
import * as _ from "litus";

_.cumsum([-4, 1, 0, 3, 2]);
// => [-4, -3, -3, 0, 2]
```
