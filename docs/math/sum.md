# sum

Calculates the sum of an array of numbers

`Since v1.0.0`

## Arguments

- `arr: number[]` - Numbers to sum

## Returns

`number` - Sum of all numbers in the array

## Usage

```ts
import * as _ from "litus";

_.sum([-4, 1, 0, 3, 2]);
// => 2
```
