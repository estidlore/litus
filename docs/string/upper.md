# upper

Converts a string to UPPERCASE

`Since v1.8.0`

## Arguments

- `str: S` - String to convert

## Returns

`Uppercase<S>` UPPERCASE version of the input string

## Usage

```ts
import * as _ from "litus";

_.upper("Hello World!");
// => 'HELLO WORLD!'
```
