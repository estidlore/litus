# lower

Converts a string to lowercase

`Since v1.8.0`

## Arguments

- `str: S` - The string to convert to lowercase

## Returns

`Lowercase<S>` The lowercase version of the input string

## Usage

```ts
import * as _ from "litus";

_.lower("Hello World!");
// => 'hello world!'
```
