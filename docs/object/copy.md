# copy

Creates a deep copy of the provided object or array

`Since v1.0.0`

## Arguments

- `obj: T`

## Returns

`T`

## Usage

```ts
import * as _ from "litus";

_.copy({ age: 18, name: "John" });
// => { age: 18, name: "John" }
```
